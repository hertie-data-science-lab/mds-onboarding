This preparation package contains recommended materials for incoming candidates to review the concepts and fundamentals in **programming** and **mathematics**. Please go through it carefully so you can get the best possible start to the programme. There is no need to rush, and you are not expected to master everything — the aim is to arrive confident with the basics.
